## 🤝 정리 규칙 & 협업 방식

1. **개인 브랜치로 정리하기**
   - 브랜치 이름은 `정리자 이름`을 기준으로 생성: 예) `feature/heechan-study`
   - 해당 브랜치에서 자유롭게 정리 및 커밋


2. **디렉터리 구조**
   - 주제별로 디렉터리를 나누고, 그 안에 `README.md`로 정리
   - 예제 코드가 있다면 `/code` 하위에 작성


3. **메인에 반영하기**
   - 정리를 완료하고 스터디를 통해 적합한 내용을 모아 main에 반영
